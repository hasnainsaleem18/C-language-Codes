    <h1>C++ Programs</h1>

    <h2>1. Find the Largest Integer:</h2>
    <p>Reads two integers from the user. Outputs the largest one using if-else statements.</p>
    <code>largest_integer.cpp</code>

    <h2>2. Check Even or Odd:</h2>
    <p>Reads an integer from the user. Outputs whether the number is even or odd using if-else.</p>
    <code>even_odd_checker.cpp</code>

    <h2>3. Find the Largest Among Three:</h2>
    <p>Reads three integers from the user. Outputs the largest one.</p>
    <code>largest_among_three.cpp</code>

    <h2>4. Attendance Check for Exam Eligibility:</h2>
    <p>Asks the user for the total number of classes scheduled and attended. Passes these values to a function <code>calculateAttendance</code>. Calculates attendance, displays it, and decides if the student is allowed to sit in the exam.</p>
    <code>attendance_check.cpp</code>

    <h2>5. Leap Year Checker:</h2>
    <p>Asks the user to enter a year. Passes the year to the function <code>isLeap</code> which checks if it's a leap year or not. Prints the result.</p>
    <code>leap_year_checker.cpp</code>

    <h2>6. Cookie Shipping Calculation:</h2>
    <p>Prompts the user to enter the total number of cookies, cookies in a box, and boxes in a container. Outputs the number of boxes and containers needed to ship the cookies. Handles leftover cookies and boxes.</p>
    <code>cookie_shipping.cpp</code>

    <h2>7. Earthquake Magnitude Descriptor:</h2>
    <p>Reads magnitude from the user. Displays a meaningful message about the earthquake magnitude.</p>
    <code>magnitude_descriptor.cpp</code>

    <h2>8. Largest Odd Number Checker:</h2>
    <p>Examines three variables (x, y, z). Prints the largest odd number among them or a message if none are odd.</p>
    <code>largest_odd_checker.cpp</code>
